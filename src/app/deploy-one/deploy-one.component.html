Step 1) Push your all files into github as normal

In this lesson I will tell you how to how to deploy your angular project in github pages and view as a client Firstly, create a new repository. Make it public
Now open git bash inside the folder you had all of the files which you want to make remote.
Initialize local git repository
git init
Add remote repository URL to local(angular-one = sample git repository name in my case)
git remote add origin https://github.com /ShristySk1/angular-one.git
Add all files in the local git repository (You must consider the space after add)
git add .
Commit to the local machine
git commit -m "Initial commit"
Push into the remote repository
git push -u origin master

Step 2) Now Deploying in server(We do these in gitbash or internaL terminal of visual studio code where we wrote our code)

now in git bash or internal terminal of visual studio code, write the code as below
npm install -g angular-cli-ghpages
For generating a dist folder for production
ng build --prod --base-href "https://shristysk1.github.io/angular-one/"
Now upload the files and folders inside dist created
ngh --dir dist/clientxx
in my case there was subfolder clientxx so i have to go inside it at last otherwise---> ngh --dir/dist
PS: Repository in github must be public
