Firstly, create a new repository. Make it private or public according to your choice. You will get into this kind of screen like below:
Now open git bash inside the folder you had all of the files which you want to make remote.
Initialize local git repository
git init
Add remote repository URL to local
git remote add origin https://github.com /ShristySk1/test.git
Add all files in the local git repository (You must consider the space after add)
git add .
Commit to the local machine
git commit -m "Initial commit"
Push into the remote repository
git push -u origin master
